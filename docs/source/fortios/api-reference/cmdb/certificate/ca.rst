Ca
==

Configuration endpoint for certificate/ca.

Python Attribute
----------------

.. code-block:: python

   fgt.api.cmdb.certificate.ca

Available Methods
-----------------

- ``get()`` - GET operation

Examples
--------

.. code-block:: python

   from hfortix_fortios import FortiOS
   
   fgt = FortiOS(host='192.168.1.99', token='your-token')

   # List all items
   items = fgt.api.cmdb.certificate.ca.get()

   # Get specific item by name
   item = fgt.api.cmdb.certificate.ca.get(name='item-name')
