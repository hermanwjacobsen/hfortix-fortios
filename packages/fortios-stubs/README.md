# hfortix-fortios-stubs

Type stubs for the `hfortix-fortios` package.

## What are type stubs?

Type stubs (`.pyi` files) provide type information for Python code, enabling:
- Better IDE autocomplete and IntelliSense
- Static type checking with tools like mypy, pyright
- Enhanced code documentation and type hints

## Installation

This package is automatically installed when you install `hfortix-fortios`:

```bash
pip install hfortix-fortios
```

Or install manually:

```bash
pip install hfortix-fortios-stubs
```

## Usage

No additional setup required. Once installed, your IDE and type checkers will automatically use these stubs.
